# DreamAgent 마케팅 사이트 — CLAUDE.md

## 무엇을 만드나
DreamAgent(도메인별 AI 에이전트 개발사)의 **제품 주도 마케팅 쇼케이스 사이트**.
실제 로그인·제품 진입 없음. 데모는 "연출". 모든 CTA는 "데모 요청"으로 수렴.

## 작업 전 반드시 읽을 것
- `docs/컨셉-정의서.md`   — 마스터 스펙 (홈 8단 흐름 · 라인업 · 차별 3축)
- `docs/슬로건-정의서.md` — 메시징 · 카피 기준
- `docs/기획안.md`        — IA · 섹션 카피 · 컴포넌트 아키텍처

## 스택
- Next.js (App Router, SSG)
- 섹션 레지스트리 패턴: 페이지 = 섹션 컴포넌트 배열. 새 제품 = `content/products` 데이터 1개 추가로 자동 생성.
- 문자열은 전부 i18n 키로 분리 (한국어 우선, 영어 토글 여지 — 컴포넌트에 하드코딩 금지)
- Tailwind + 시맨틱 토큰 레이어 (구체 색·폰트는 `design.md` 확정 전까지 placeholder 값)

## 슬로건 (고정)
"이제 전문가는, 당신입니다." — 핵심은 **권한 이동**.
- AI를 문장의 *주어*로 올리지 말 것 ("AI가 분석해드립니다" = 대체 프레임, 금지). AI는 받쳐주는 조연.
- "쉽다/편하다"를 직접 헤드에 쓰지 말 것 — 결과(권한 이동)로 말한다.

## 디자인 규칙 (don't)
- AI 슬롭 금지: 보라/파랑 그라데이션 · 떠다니는 blob · 과도한 glass · 일률 라운드
- 추상 이미지로 제품 가리지 말 것 — **실제 제품 화면이 주인공**
- 한글 우선, **Pretendard**
